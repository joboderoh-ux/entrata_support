The Problem

Support engineers waste time hunting for the same links every day

Entrata's support teams rely on a wide variety of internal tools, documentation pages, and resources spread across bookmarks, Slack messages, pinned tabs, and tribal knowledge. There is no single, consistent location where support engineers can find the links most relevant to their team. New engineers take longer to ramp up, experienced engineers lose time context-switching to find resources, and when links change there is no central place to update them.

Impact

A small fix with daily impact across the entire support org

9
Product support teams covered in the app
0
Tab switching or searching required — links live inside Zendesk
1
Place to update a link — changes reflect instantly for the whole team
The Solution

One app, inside Zendesk, always where engineers need it

The Quick Links App lives in the Zendesk sidebar — visible on every ticket without switching tabs. It displays all 9 support team tiles. Engineers click their team, see the most commonly used links for that team, and navigate instantly. Links are stored in a database and can be added, edited, or removed without touching any code.

How It Works

Simple by design — easy to use, easy to maintain

Technical Overview

Lightweight, maintainable, no ongoing engineering required

🎫
Zendesk Sidebar App

Built as a native Zendesk app — lives inside the sidebar on every ticket. No separate tool to open or remember.

🗄️
Supabase (Database)

Links are stored in a database. Add, edit, or delete links anytime without touching code. Changes are instant.

📄
Single HTML File

The entire app is one HTML file. Simple, fast, and easy to update if the UI ever needs changing.

🔧
manifest.json

Standard Zendesk app configuration file that registers the app and defines where it appears in the UI.
